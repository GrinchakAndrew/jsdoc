<static> {String|null} mmcore.CGRequest(callback)
Makes the request. The workflow is:
  • Read the cookie 'mmact'
  • Deserialize the cookie value
  • Call SetAction for each deserialized item
  • Clear and delete cookie 'mmact'
  • Increment mmcore._request_id
  • Update the mmcore._callback by the callback
  • Appends the head/body by script/pixel correspond to actions
The callback is called from the loaded script (seee the last point of the workflow)
Parameters:
{Function} callback
The callback function
Returns:
{String|null}

<static> mmcore.HideMaxyboxes(names)
Hides the maxyboxes by their ID's. The function appends the <head> by the style block like follow:
#boxId1,#boxId2,#boxId3 {
    visibility:hidden;
}
------- The function is called from the server response stuff (the first answer)
The function is called just after the renderers set, before the call of _RenderOnLoad
Parameters:
{String} names
The list of maxyboxes ID's. May be set as an Array: 
mmcore.HideMaxyboxes(['boxId1','boxId2','boxId3']);
or as an comma separated list: 
mmcore.HideMaxyboxes('boxId1','boxId2','boxId3');
<static> mmcore.IsDefaultArrived(m)
The function is checking if the maxybox DOM element appears in the page DOM. The searching of maxybox execs using the function mmcore.GetMaxyboxNode;
The function can be used at any moment (we don't need event 'onready') because the function additionally checks, if the parent node is ready, or, maybe, the maxybox has the next DOM block
Parameters:
{String} m
The maxybox ID
Returns:
true when the default content appears

<static> mmcore.RenderMaxyboxes(names)
Renders maxyboxes. When the maxyboxes list is set (through names) the function renders just set maxyboxes. Otherwise it calls all renders.
The names should met the company elements names.
After the maxyboxes rendered the function ShowMaxyboxes is called.
-------
The completed list of render functions is stored in mmcore._renderers
To avoid the multiple call of a renderer there is the list of flags mmcore._r_mbs. When renderer was called the corresponding flag is set to 1.
In another words, when flag is set to 1 the corresponding renderer will not be called.
Parameters:
{String|Array} names
{Optional} List of maxyboxes names to be rendered
